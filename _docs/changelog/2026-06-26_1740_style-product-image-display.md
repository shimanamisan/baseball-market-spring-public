# 2026-06-26 17:40 商品画像表示のメルカリ準拠統一とレビュー指摘対応

> ブランチ `feature/product-image-display`（PR #85）。商品画像の見せ方をメルカリに準拠させ、一覧/マイページのグリッドを 1:1 正方形・中央クロップに統一、商品詳細のメイン画像は contain + ブラー背景フィルで全体表示にした。あわせて Gemini Code Assist のレビュー指摘（Issue #86）に対応し、画像未登録時のプレースホルダー追加とサブ画像 alt 設定を行った。純フロント（Thymeleaf テンプレート）変更のみ。

## 背景

商品画像のアスペクト比がページ間でバラバラで、見切れ（クロップ）や枠の余白が不統一だった。メルカリの見せ方を基準に、一覧・マイページ・詳細で表示ルールを揃える。

## 変更内容

### 1. 画像表示のメルカリ準拠統一 — commit b3733e7

- **一覧 (`product/list.html`)**: `aspect-square` と競合していた固定高さ `h-64` を削除して真の 1:1 正方形に。`object-top`（上寄せ）→ `object-center` で中央クロップへ変更。
- **マイページ (`mypage/index.html`)**: 出品/お気に入りグリッド 2 箇所も冗長な `h-48` を削除し 1:1 正方形に統一（中央クロップは既存挙動と一致）。
- **商品詳細 メイン画像 (`product/detail.html`)**: 同一画像を 2 枚重ね、背景を `object-cover blur-2xl scale-110` でボカして枠を埋め、前景を `object-contain` で全体表示。背景は装飾のため `aria-hidden`、前景に商品名 `alt` を付与。
- **商品詳細 サムネイル**: `object-cover` → `object-contain bg-gray-100` で見切れなしに統一。
- `saleHistory.html` は元から固定正方形 + 中央クロップで方針一致のため変更なし。

### 2. レビュー指摘対応（Gemini #86・いずれも妥当） — commit 6271ced

Gemini Code Assist のインライン指摘 2 件を検証し、いずれも妥当と判断して対応（`product/detail.html`）。

- **メイン画像のフォールバック追加・条件重複の解消**: `product.pic1` 未登録（null/空）時に空のグレーボックスになっていたため、一覧 (`list.html`) と同じ `ri-image-line` プレースホルダーを `th:unless` で表示。あわせて 2 枚の `<img>` に重複していた `th:if` 条件を `<th:block>` に集約。
- **サブ画像の alt 設定**: サムネイル (`pic2`/`pic3`) の `alt=""`（装飾扱い）→ `th:alt="|${product.productName} のサブ画像|"` に変更し、コンテンツ画像としてアクセシビリティを改善。
