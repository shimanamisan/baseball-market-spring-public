# baseball-market-spring

野球グッズマーケット（旧 baseball-market / PHP 版）を Spring Boot でフルリプレースするリポジトリ。
Devcontainer により Spring Boot の開発環境を構築する。

## 環境構成
- Java 21 (Bullseye)
- Gradle
- MySQL 8.0
- Spring Boot Dev Tools

## 使用方法
1. VS Codeとリモートコンテナ拡張機能をインストールします
2. このリポジトリをクローンします
3. VS Codeでフォルダを開き、「Reopen in Container」を選択します
4. コンテナが起動すると、Java開発環境とMySQLデータベースが利用可能になります
